Welcome to IMLS Labs, a place for visitors to try new capabilities IMLS is exploring for sharing information with the public. Some of these capabilities may still have bugs in them—please help us find those issues and send us feedback.

IMLS Labs hosts recent new data exploration tools we are seeking feedback about. The latest is the Library Search & Compare, a tool which provides an easy way to query and view data from the FY2016 Public Libraries Survey. All data comes from IMLS FY2016 Public Libraries Survey.
